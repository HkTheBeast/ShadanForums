What is Svlent?

A Digital Monitor
for Every Classroom

Svlent is a teacher's digital assistant — a web-based classroom management platform that replaces scattered spreadsheets, notebooks, and manual tracking. Built from scratch using modern web technologies, it gives teachers a clean, unified interface to manage every aspect of their class.

Originally integrated into the ShadanForums platform, Svlent is a standalone system with its own secure teacher login, isolated data per teacher, and a beautiful dark UI designed to work on any device.

Core Modules

Everything a Teacher Needs

Attendance Tracker

Mark each student as Present, Late, or Absent for any date with one tap. View daily stats — total present, absent, late — at a glance. Switch to Summary mode to see full attendance history per student with percentage breakdowns. The system calculates attendance rates automatically.

Daily View Summary View Auto % Calc Date Picker

Submission Tracker

Track 5 submission types per student — Assignment 1, Assignment 2, Record Book, Observation Book, and PPT. Simple on/off toggles with instant save. Colour-coded status badges and summary stats at the top.

5 Types Tracked

Marks & Grades

Enter marks for up to 4 assessments per student. The system auto-calculates totals, assigns letter grades (S/A/B/C/D), and shows score rings and metric bars on each student's profile card.

Auto Grading

Highlights & Warnings

Recognise outstanding students with a gold highlight, or issue up to 3 red warning dots for behavioural or academic concerns. When a student hits 3 warnings the badge pulses as an alert.

Filter the list to see only highlighted or warned students instantly. Warning counts show on the student's profile card across all pages.

Gold Stars 3-dot Warnings Pulse Alert

Student Profiles

Each student has a rich profile card showing avatar, roll number, grade badge, score ring, metric bars for attendance/submissions/marks/behaviour, and quick-nav buttons to every module.

Full Profiles

Teaching Planner

Three powerful tools in one: Progress Trackers (custom metric rings with steppers), Curriculum Builder (subjects → topics → subtopics with completion checkboxes), and a Weekly Timetable grid.

3-in-1 Tool

Built with Modern Web Technology

Svlent is a full-stack web application built entirely from scratch by a student development team. The backend is Node.js + Express with a SQLite3 database. The frontend is pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean code. Sessions are managed server-side for security. File uploads (avatars) are handled with Multer and stored in the server filesystem.

Teaching Planner

Plan Your Entire
Teaching Year

The Teaching Planner is the most powerful module in Svlent. It combines three tools that every teacher needs but rarely has in one place.

Progress Trackers Create custom trackers for anything — classes completed, syllabus %, chapters done. Each tracker has a progress ring, stepper controls, and a colour theme.
Curriculum Builder Organise subjects into topics and subtopics. Mark each item done as you teach. Filter by section. See real-time completion percentages per subject.
Weekly Timetable Build your weekly schedule visually. Add colour-coded class slots for each day with time, subject, section, and room. Edit or delete with one click.
