Q1 :Give a detailed overview of digital computers,including their key characteristics and application

Answer:

a digital computers is an electronic system that processes information using binary digits (0&1).it accepts data,performss operations according to instructions,stores intermediate results, and generates output.

KEY CHARACTERISTICS :
  • works on binary system (0&1)
  • high speed & accuracy
  • programmable & versatile
  • large storage capacity
  • automation and reliability
APPLICATION:
  • scientific research (simulations,weather forecasting).
  • (payroll,banking,accounts).
  • (CAD tools,circuit design).
  • medical (diagnosis,imaging).
  • education & communication(online learning,networking).

SUMMARY:

Digital computers are fast,accurate and versatile machines used across science,business,engineering and daily life

Q2 : Explain bus system for four registers

ANSWER:

A bus system provides a common pathway for data transfer among components in a system with four registers(R1-R4).

  • MULTIPLEXER:

    Selects which register sends data to the bus

  • Decoder:

    Activates the correct destination register to recieve data

  • CONTROL LINES:

    manage read /write operation

SUMMARY:

A bus system reduces hardware complexity while allowing multiple registers to share a single data path.

Q3 Explain the following terms.

  • a) Desing of control unit
  • b) Compare between hardwired control unit & microprogram control unit
a)CONTROL UNIT DESIGN:

The CU directs operations in the CPU by generating control signals. inputs include the instruction register, flag, and clock.outputs are signals and ALU, memory & input devices

b) COMPARISON:
  • HARDWIRED CU:

    implementented with combinational logic, veryfast but inflexible, used in RISC processors

  • MICROPROGRAMMED:

    Uses microinstructions stored in control memory, slower but flexible, easy to modify, used in CISC processors

SUMMARY:

Hardwired CU= fast but rigid;
microprogrammed CU=flexible but slower
