Answer :

A bus system provides a common pathway for data transfer among components in a system with four registers(R1-R4).

  • MULTIPLEXER:

    Selects which register sends data to the bus

  • Decoder:

    Activates the correct destination register to recieve data

  • CONTROL LINES:

    manage read /write operation

SUMMARY:

A bus system reduces hardware complexity while allowing multiple registers to share a single data path.

  • b) Compare between hardwired control unit & microprogram control unit
a)CONTROL UNIT DESIGN:

The CU directs operations in the CPU by generating control signals. inputs include the instruction register, flag, and clock.outputs are signals and ALU, memory & input devices

b) COMPARISON:
  • HARDWIRED CU:

    implementented with combinational logic, veryfast but inflexible, used in RISC processors

  • MICROPROGRAMMED:

    Uses microinstructions stored in control memory, slower but flexible, easy to modify, used in CISC processors

SUMMARY:

Hardwired CU= fast but rigid;
microprogrammed CU=flexible but slower

Answer :

Addressing modes defines how operands are accessed in instructions

  • IMMEDIATE:operands in instruction
  • EXAMPLE:MOV A, #5
  • DIRECT:Address of operand given in instruction
  • EXAMPLE:MOV A, 1000
  • INDIRECT:Instruction pointd to memory location containg operands address
  • EXAMPLE;MOV A, (R1)
  • REGISTER:Operand in register
  • EXAMPLE:ADDA, R1
  • indexedEffective address=base+index register
  • EXAMPLE:MOV A, 1000(R2)
  • RELATIVE:Address relative to PC
  • EXAMPLE:JMP 20
    • SUMMARY:Addressing modes provide flexible ways to access operands during instruction execution

Floating point representation is used to store real numbers ( fractions, very large, or small values).

  • GENERAL FORM: N=(-1)³xMx2⁶
  • IEEE 754 SINGLE PRECISION (32-BITS):
  • 1 bit:sing(S)
  • 8 bits:Exponent(E), bias=127
  • 23 bits:mantissa(M)
    • EXAMPLE:5.75->Sign=1, exponent in biased form , mantissa is binary fraction.
SUMMARY:Floating point allows storage of very large/small real numbers efficiently.
