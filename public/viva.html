OBJECT ORIENTED PROGRAMMING THROUGH JAVA :


QUESTION:

1.What is Java?

ANS: Java is an object-oriented, platform-independent programming language used to build applications.


2.Who developed Java, and in which year?

ANS: The four main features are: PEncapsulation ,Abstraction ,Inheritance ,Polymorphism


3.What are the main features of Java?

ANS: Simple, Object-Oriented, Platform Independent, Secure, Robust, Portable, and Multithreaded


4.What is JVM, JRE, and JDK?

ANS: JVM (Java Virtual Machine): Runs Java bytecode.

JRE (Java Runtime Environment): Contains JVM and libraries to run Java programs.

JDK (Java Development Kit): Contains JRE + tools to develop Java programs.


5.What is bytecode?

ANS: Bytecode is an intermediate code generated after compiling Java code. It is executed by JVM.


6.Why is Java platform independent?

ANS: Because Java programs are compiled into bytecode which can run on any system having JVM.


7.What is the difference between Java and C++?

ANS: Java is platform-independent and has no pointers; C++ is platform-dependent and supports pointers.


8.What is a class and an object?

ANS: Class: Blueprint or template of an object.

Object: Instance of a class.


9.What is the difference between a constructor and a method?

ANS: Constructor: Used to initialize an object; same name as class.

Method: Used to perform operations; can have any name.


10.What is the use of the new keyword?

ANS: It is used to create an object in Java.

R LANGUAGE:

1.What is R?

ANS: R is an open-source programming language and software environment for statistical computing and graphics.


2.Who developed R and when?

ANS: R was developed by Ross Ihaka and Robert Gentleman at the University of Auckland in 1993


3.What are the main features of R?

ANS : Open-source, platform-independent, statistical analysis, graphics, data visualization, and extensible with packages.


4.What is R used for?

ANS: Data analysis, statistical computing, data visualization, machine learning, and reporting.


5.What is the difference between R and Python?

ANS: R is mainly for statistics and data analysis; Python is general-purpose and widely used in programming, web development, and data science.


6.How do you install R and RStudio?

ANS:Install R from CRAN Install RStudio IDE to write and run R scripts easily.


7.What is CRAN?

ANS: CRAN (Comprehensive R Archive Network) is the official repository of R packages and source code


8.What is the difference between R and RStudio?

ANS:R: Programming language and environment. RStudio: IDE for writing and running R scripts


9.What are vectors in R?

ANS: A vector is a one-dimensional array that holds elements of the same type (numeric, character, or logical).


10.What are data types in R?

ANS:Numeric, Integer, Character, Logical, Factor, Complex, and Raw.
